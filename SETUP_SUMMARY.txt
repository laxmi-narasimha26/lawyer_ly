================================================================================
  TROUBLESHOOTING:
================================================================================

Problem: Docker not running
Solution: Start Docker Desktop and wait for it to fully start

Problem: Backend won't start
Solution: cd backend && pip install -r requirements.txt

Problem: No AI responses
Solution: Check OPENAI_API_KEY is set correctly

Problem: Database connection error
Solution: docker restart legal-ai-postgres-local

Problem: Port already in use
Solution: Kill the process using that port or change the port
